# Apptics Form - Simple Swap Component

A minimal Next.js application featuring a simple cryptocurrency swap interface.

## Features

- Simple swap form (ETH ↔ USDT)
- Real-time price calculation
- Clean and minimal UI
- Built with Next.js 15, React 19, and Tailwind CSS

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

```
src/
├── app/
│   ├── page.tsx          # Main page using SimpleSwap
│   ├── layout.tsx        # Root layout
│   └── globals.css       # Global styles
└── components/
    └── SimpleSwap.tsx    # Main swap component
```
